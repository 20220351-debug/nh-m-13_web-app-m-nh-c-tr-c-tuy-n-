# nhom 13_web-app am nhac truc tuyen
web app nghe nhạc trực tuyến là nền tảng cho phép người dùng nghe nhạc ngay trên trình duyệt mà không cần tải file về máy. Các dịch vụ phổ biến như Spotify hay YouTube Music đều hoạt động dựa trên mô hình này.

# 🎵 Nhóm Flutter App

## 👥 Thành viên
- Phạm Thùy Dương - Trưởng nhóm
- Nguyễn Lê Thùy Linh - Thành viên

## 📱 Mô tả
Ứng dụng nghe nhạc trực tuyến xây dựng bằng Flutter : Ứng dụng cho phép tạo playlist, nghe nhạc với các chức năng play/pause, chế độ offline mode cho phép quét file nhạc trong thiết bị, tạo danh sách từ file nhạc trên máy


## 🚀 Công nghệ
- Flutter
- Dart
